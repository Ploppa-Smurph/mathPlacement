12.01 - Adding and Subtracting Polynomials

    - Review combining like terms
                ex: -4p^7 + 8p^7 + 5p^9
                    = 4p^7 + 5p^9
                    = 5p^9 + 4p^7

                ex02: -4y^2 + 3y^2 - 2y^2 + y^2 
                    = -2y^2

    - Know vocabulary for polynomials 
                a polynomial in x is a term, or a finite sum of terms, of the form ax^n,
                  for any real number a and any whole number n.

                a polynomial is written in descending powers (higest power first) from left to right.
                    ex: 6x^8 - 7x^6 + 5x^4 - 3x^2 + 4
                        you sort based in descending order
                        
                the degree of a term is the sum of the exponents on the variables

                the degree of a polynomial is the highest degree of anynonzero term of the polynomial
                    ex: (2x^2)*y = 2x^2*y has a degree of 3 --> 2x's + 1y = 3
                    ex02: 3x^4 - 5x^2 + 6 has a degree of 4 --> since this is a polynomial - you use the highest degree and it is 4 for this example

                monomial = polynomial with only a single (1) term:
                    ex: 9m or -6y^5

                binomial = polynomial with exactly 2 terms: 
                    ex: -9x^4 + 9x^3 or 8m^2 + 6m

                trinomial = polynomial with exactly 3 terms: 
                    ex: 9m^3 - 4m^2 + 6   -- 3 terms             
            
            ex: 6x^4 - 9x  --> this is a binomial that is already in descending order
                                         the degree for this binomial is 4 (the highest exponent)
                                       
            ex02: 5m^4 - 3m^2 + 6m^5 - 7m^3 
                    6m^5 + 5m^4 - 7m^3 - 3m^2  --> this has more than 3 terms and is simply polynomial (this has 4 polynomials)
                                                    the degree for this polynomial is 5 (the highest exponent)                            

                

    - Evaluate polynomials
        ex: find the value of 3x^4 + 5x^3 - 4x - 4 if x = -2
               3(-2^4) + 5(-2^3) - 4(-2) - 4
               3*16 + 5(-8) + 8 - 4
               48 - 40 + 8 - 4 = 12

        ex02: find the value of the polynomial
                2x^2 + 5x + 1 when x = 2
                2*(2)^2 + 5(2) + 1
                2(4) + 10 + 1 = 19    -- this is a trinomial with a degree of 2 

    - Add polynomials
        vertical
        ex: add 6x^3 - 4x^2 + 3 to 
              + -2x^3 + 7x^2 -5 
         answer 4x^3 + 3x^2 -2
        
        horizontal
        ex: add 2x^2 - 4x + 3 and + x^3 + 5x
                (2x^2 - 4x + 3) + (x^3 + 5x)
                x^3 + 2x^2 + x + 3
                
    - Subtract polynomials
        ex: subtract 9x^2 - 3x + 7 from -2x^2 - 6x + 4
            (-2x^2 - 6x + 4) 
           - (9x^2 - 3x + 7)  
           -----------------
           -11x^2 - 3x - 3

    - Add and subtract polynomials with more than 1 variable 
        ex: (2c^4d + 3c^2d^2 - 4d^2) - (c^4d + 8c^2d^2 - 5d^2)
               c^4d - 5c^2d^2 + d^2 


12.02 - the Product Rule and Power Rules for Exponents with Tom Atwatter

    - Use exponents
        ex: write (1/2)(1/2)(1/2)(1/2)(1/2)(1/2) as exponents:
             (1/2)^6

        ex02: 3^5;   identify base: 3; identify exponent: 5 
        evaluate: 3^5 = 243 
                  
        ex03: -3^5   identify base: -3; identify exponent: 5
        evaluate: -3^5 = -243

    - Use the product rule for exponents
        for any poistive integers m and n
            a^m * a^n = a^mn   -- keep the same base and add the exponents
        
        ex: 5^2 * 5^6 = 5^2+^6 = 5^8    since the base is the same you keep it and add the exponents together 

        ex02: t^3 * t^8 * t^13 = t^24

        ex03: -6p^5 * -7p^5 = 42p^10     since the base numbers are different you do the math. 

        ex04: 3^8 + 3^9 = addition problem does not work -- have to expand each exponent
              evaluate: 6561 + 19,683 = 26,244

    - Use the rule (a^m)^n = a^mn
        for any positive integer m and n 
         (a^m)^n = a^mn   you raise a power to a power by multiplying the exponents
         
         ex: (4^3)^2 = 4^6
         
         ex02: (t^4)^5 = t^20

    - Use the rule (ab)^m = a^m * b^m
        for any positive integer m and n
            (ab)^m = a^m * b^m   raise a product to a power by raising each factor to the power
            Product = the answer to a multiplication problem
            
         ex: (5xy)^5 = 5^5 * x^5 * y^5

    - Use the rule (a/b)^m = a^m/b^m 
        for any postive integer m, 
            (a/b)^m = a^m/b^m   raise a quotient to a power by raising both numerator and denominator to the power
            Quotient = number obtained by dividing one quantity by another

         ex: (1/2)^3 = 1^3/2^3 = 1/8  

    - Use combinations of rules
         ex: (9/8)^3 * 9^2
             (9^3 / 8^3) * 9^2
             9^3 * 9^2 / 8^3
             9^5 / 8^3
             evaluates to: 59,049 / 512

         ex02: (-r^4 * s)^2 * (-r^2 * s^3)^5
               (r^8 * s^2) * (-r^10 * s^15) 
               -r^18 * s^17

    - Use the rules of exponents in a geometry application
        ex: find the expression that represents the area of the figure drawn
             (it is a light blue rectangle on a dark blue field) - measurements given
              3x^2 = 1 side width and 4x^3 = 1 side length
              find the area by multiplying length times width

              A = 4x^3 * 3x^2 = 12x^5
              A = 12x^5        -- since the x was the same base with can add the exponents        
  

12.03 - Multiplying Polynomials

        - Multiply a monomial and a polynomial
            ex: find the product of: 2m(3m + 2)
                 2m(3m) + 2m(2) 
                 6m^2 + 4m     

        - Multiply two polynomials
            to multiply 2 polynomials multiply each tearm of the 2nd polynomial by each term of the first polynomial and add the products
            ex01: (6x + 1)(2x^2 + 4x + 1)     ==> binomial multiplied by trinomial
                  12x^3 + 24x^2 + 6x
                        + 2x^2  + 4x + 1
                  ----------------------
                  12x^3 + 26x^2 + 10x + 1          

            ex02: (4m + 3)(5m^3 - 4m^2 + m - 5)
                  20m^4 - 16m^3 + 4m^2 - 20m
                        + 15m^3 -12m^2 + 3m -15
                  -----------------------------
                  20m^4 - m^3 - 8m^2 - 17m - 15   
                  
            ex03: (6x^4 - 4x^2 + 8x)(1/2x + 3)
                  (6x^4)(1/2x) + (6x^4)(3)
                + (- 4x^2)(1/2x) + (-4x^2)(3)
                + (8x)(1/2x) + (8x)(3)
                ------------------------------
               3x^5 + 18x^4 - 2x^3 - 12x^2 + 4x^2 + 24x
               3x^5 + 18x^4 - 2x^3 -8x^2 + 24x 

        - Multiply binomials by the FOIL (First, Outer, Inner, Last) method
           Step 1: Multiply the First 1st terms of the binomials to get the first term of the answer
           Step 2: Find the Outer product and Inner product and add them (when possible) to get the middle term of the answer
           Step 3: Multiply the 2 Last terms of the binomials to get the Last term answer  

            ex: (x + 3)(x + 5)
                F: x^2
                O: 5x
                I: 3x
                L: 15

            ex02: (m + 7)(m + 5)
                F: m^2
                O: 5m
                I: 7m
                L: 35
                m^2 + 12m + 35
            
            ex03: (4x + 3)(2y - 1)
                  8xy - 4x + 6y - 3  

            ex04: (3x + 2y)(5x - 3y)
                  15x^2 - 9xy + 10xy -6y^2
                  15x^2 + xy - 6y^2 
         

12.04 - Special Products

                shortcuts to find certain binomial products

        - Square binomials
                (x + y)^2 = x^2 + 2xy + y^2
                also (x - y)^2 = x^2 - 2xy + y^2

            ex: (2x + 3)^2
                  
                   a) what is the square of the first term? 
                      -- 4x^2

                   b) what is twice the product of the two terms?  
                      -- 2(2x)(3) = 12x

                   c) what is the square of the last term?
                      -- (3)^2 = 9
                    
                   d) write the final product (a trinomial) using the results of a) - c)
                      -- 4x^2 + 12x + 9  

            ex02: (x + 2y)^2
                  x^2 + 4xy + 4y^2      
 
            ex03: (6m - 4/5n)^2
                  36m^2 - 48/5mn + 16/25n^2                          

        - Find the product of the sum and difference of two terms
            (x + y)(x - y) = x^2 - y^2
            
            ex: (k + 5)(k - 5)
                = k^2 - 25

            ex02: (3/4 - x)(3/4 + x)
                = 9/16 - x^2
            
            ex03: (10x + 3y)(10x - 3y)
                  100x^2 - 9y^2  ??? is the answer -- doesn't follow the rules to this point ???

            ex04: q(5q-1)(5q+1) 
                  q(25q^2 - 1)
                  25q^3 - q  
            
        - Find greater powers of binomials 
            ex01: (x + 1)^3
                  (x + 1)(x + 1)^2
                  (x + 1)(x^2 + 2x + 1)
                  x^3 + 2x^2 + x + x^2 + 2x + 1
                  x^3 + 3x^2 + 3x + 1 

            ex02: (4x - 1)^4
                  (4x - 1)^2 * (4x - 1)^2
                  (16x^2 - 8x + 1)(16x^2 -8x + 1)
                  256x^4 - 128x^3 + 16x^2 - 128x^3 + 64x^2 -8x + 16x^2 -8x + 1
                  256x^4 - 256x^3 = 96x^2 - 16x + 1  
            

12.05 - Integer Exponents and the Quotient Rule

        - Use 0 as an exponent
                you get 1 as an answer, as long as a is != 0

                a^0 = 1

              ex:    60^0    = 1
              ex02:  (-60)^0 = 1
              ex03:  -60^0   = -1
              ex04:  y^0     = 1 if y != 0
              ex05:  6y^0    = 6 if y != 0
              ex06:  (6y)^0  = 1 - if y != 0

        - Use negative numbers as exponents
                for any nonzero real number 'a' and any integer n
                a^-n = 1/a^n

              ex: 3^-2 = 1/3^2
                  1/9

              ex02: (2/5)^-4 = 5/2^4
              
              ex03: 4^-1 - 2^-1 = 1/4 - 1/2 
                    -1/4

              ex04: 1/x^-4 = x^4    
              
            ** A negative exponent DOES NOT indicate a negative number; negative eponents lead to reciprocals   

              ex: a^-n        3^-2 = 1/3^2 = 1/9     not a negative

              ex02: -a^-n    -3^-2 = -1/3^2 = -1/9   is negative
        
        changing from negative to positive exponents
           for nonzero numbers a and b and any integers m and n 
           
           a^-m/b^-n = b^n/a^m and (a/b)^-m = (b/a)^m
              
              ex:  m^-5/p^-1
                   p^1/m^5

              ex02: a^-2b/3d^-3
                    bd^3/3a^2
                                   

              ex03: (x/2y)^-4 
                    (2y/x)^4 = (2^4 * y^4) / x^4
                     
            ** Caution: cannot change negative exponents to positve exponents using this rule if the exponents occur in a sum of terms

              ex: (5^-2 + 3^-1) / 7 - 2^-3  cannot be flipped and rewritten like above because it contains addition and subtraction, not multiplication
                    you have to treat all terms as individual terms and solve it that way
                  (5^-2 + 3^1) / 7 - 2^-3  
                  (1/5^2 + 1/3) / 7 - (1/2)^3

        - Use the quotient rule for exponents
            a^m / a^n = a^m-n
            a common error is to write 5^8 / 5^4 = 1^8-4 = 1^4
           the correct answer will have the same base of 5
             5^8 / 5^4 = 5^8-4 = 5^4

              ex:  4^2 / 4^9
                   4^2-9
                   4^-7 
                   (1/4)^7 

              ex02: q^5 / q^-3
                    q^5 * q^3
                    q^8

              ex03: 3^2x^5 / 3^4x^3
                    3^2-4 * x^5-^3
                    3^-2 * x^2
                    (1/3)^2 * x^2
                    (1/9)x^2            

        - Use a combination of rules 
            ex: (2x^3 / 5)^-4
                (5 / 2x^3)^4
                5^4 / 2^4 * x^12

            ex02: (3x^-2 / 4^-1 * y^3)^-3
                 (3^-3 * x^6) / (4^3 * y^-9)
                 x^6 * y^9 / 4^3 * 3^3
                 (x^6 * y^9) / 64 * 27

            ex03: (x^-1 * y^-2 / z^2)^-2
                   x^2 * y^-2 / z^-4
                   (x^2 * z^4) / y^2 

            ex04: (m^7* n)^-2 / (m^-4n^3)
                  (m^-14 * n^-2) / m^-4 * n^3
                  m^4 / (m^14 * n^5)
                  1 / (m^10 * n^5)


12.06 - Dividing a Polynomial by a Monomial

        - Divide a polynomial by a monomial 
          ex: (2x^2 + 6) / 2 = 
               x^2 + 3
            in the ex: the Dividend is 2x^2 + 6 -- the information being divided is the dividend
                       the Divisor is 2         -- the number that is dividing into the dividend     
                       the Quotient is x^2 + 3  -- the is the result of the dividend being divided by the divisor
