16.01 Evaluating Roots

        Pythagorean Therom
      - Find Square Roots
        ex: sqrt 9
             3 & -3 are the square roots of 9 
            
        square root symbol is the 'radical' symbol,
           the number in the square root symbol is called the 'radicand'
        the square root of a number is positive, unless there is a negative before the radical symbol 
       
        ex02: sqrt 49
                7

        ex03: - sqrt 144/121 
                - 12/11

        ex04: sqrt 19 
               you square the square root and end up with the radicand as the answer
              sqrt 19^2 = 19

        ex05: sqrt (3x^2 + 4)
              sqrt (3x^2 + 4)^2
              3x^2 + 4

      - Decide whether a given root is rational, irrational, or not a real number
        ex: sqrt -25      !=  -sqrt 25
             the sqrt of a negative number is not going to be a rational number -- 
               the square root of a negative number is different from a negative square root of a radicand
        
        ex02: sqrt 25  = rational = 5
        ex03: sqrt 29  = irrational = 5.??
        ex04: sqrt -29 = not a real number -- there is no rational number that will give a - sqrt result

      - Find decimal approximations for irrational square roots
        ex: - sqrt 300     -- use calculator 
               the answer is negative 
             - 17.3205 

      - Use the Pythagorean forumula
          if C is the length of the hypotenuse of a right triangle and A and B are the lengths of the two legs
             then A^2 + B^2 = C^2

        ex: a^2 + b^2 = c^2
            8^2 + 15^2 = c^2
            64 + 225  = c^2
            c^2 = 289   
            c = 17  

        ex02: a^2 + b^2 = c^2
              a^2 + 60^2 = 100^2
              a^2 + 3600 = 10000
              a^2 = 6400
              a = 80

      - Use the distance formula
         the distance between points (x1, y1) and (x2, y2) is 
            d = sqrt (x2 - x1)^2 + (y2 - y1)^2 

        ex: find distance between (5, 7) & (1, 4)
            d = sqrt (1 - 5)^2 + (4 - 7)^2 
            d = sqrt (-4)^2  + (-3)^2
            d = sqrt 16 + 9
            d = sqrt 25
            d = 5 

      - Find cube, fourth, and other roots
       the nth root of A =   n-rt A, the n is the 'index' / 'order' of the radical

      ex: 3-rt 125 (cubed root of 125)
          = 5
      
      ex02: 3-rt -27 (cubed root of -27)
            = -3

      ex03: 4-rt 625 (4th root 625)
            = 5

      ex04: 5-rt -1024 (5th root of -1024)
            = -4


16.02 Multiplying, Dividing, and Simplifying Radicals

    - Multiply square root radicals
      Product Rule for Radicals
      sqrt A * sqrt B = sqrt AB and sqrt AB = sqrt A * sqrt B
        
        ex:  sqrt 6 * sqrt 7
             sqrt 6 *7 
             = sqrt 42

        ex02: sqrt 13 * sqrt r  (r >= 0)
              sqrt 13r     

    - Simplify radicals using the product rule
        ex: - sqrt 700
            - sqrt 100 * 7
            - sqrt 100 * sqrt 7
            -10 * sqrt 7

        ex02: 2 * sqrt 10 ** 3 * sqrt 2
              6 * sqrt 10 * sqrt 2
              6 * sqrt 20
              6 * sqrt 4 * sqrt 5
              6 * 2 * sqrt 5
              12 * sqrt 5

    - Simplify radicals using the quotient rule
      if A and B are nonnegative real numbers, and B != 0, then
             sqrt A/B = sqrt A / sqrt B and sqrt A / sqrt B = sqrt A/B
        
        ex:  sqrt 25/9
             5/3
        
        ex02: sqrt 288 / sqrt 2
              sqrt 144 * sqrt 2 / sqrt 2
              12

        ex03: sqrt 3/4
              sqrt 3 / 2

        ex04: 30 * sqrt 10 // 5 sqrt 2
                 30 / 5 = 6 
                 sqrt 10 = sqrt 5 * sqrt 2
                 sqrt 2 / sqrt 2 = 1
              6 * sqrt 5

        ex05:  sqrt 3/5 ** sqrt 1/5
               sqrt 3 * 1  / sqrt 5 * 5
               sqrt 3 / 5

    - Simplify radicals involving variables
       sqrt A^2,  for any real number A, sqrt A^2 = abslt A
       
       ex: sqrt 64x^7
           sqrt 64 * sqrt x^7
           8 * sqrt x^7
           8 * sqrt x^6 * sqrt x
           8x^3 * sqrt x

    - Simplify other roots
      nth-rt A * nth-rt B = nth-rt AB and, 
         nth-rt A / nth-rt B = nth-rt A/B
      
      ex:  3-rt 8/27
           2/3

      ex02: 3-rt p^3
            p
            
      ex03: 3-rt 16t^5
            3-rt 16 * 3-rt t^5
            3-rt 8 * 3-rt 2 * 3-rt t^5
            3-rt 8t^3 * 3-rt 2t^2
            2t * 3-rt 2t^2      


16.03 Adding and Subtracting Radicals

    - Add and Subtract radicals
           Like Radicals: 6 * sqrt 3  &&  -8 * sqrt 3
           Unlike Radicals: 7 * sqrt 6  &&  5 * 3-rt 6  -- one is sqrt and one is 3-rt

      ex: 14 * sqrt 7 -- 19 * sqrt 7      
          -5 * sqrt 7

      ex02: sqrt 17 + 4 * sqrt 17
            5 * sqrt 17

    - Simplify radical sums and differences
      ex: sqrt 45  ++  4 * sqrt 20
          sqrt 9 * 5  ++  4 * sqrt 4 * 5
          3 * sqrt 5  ++  (4 * 2) * sqrt 5
          11 * sqrt 5

      ex02: -5 * sqrt 32  ++  2 * sqrt 98
            -5 * (sqrt 16 * sqrt 2)  ++  2 * (sqrt 49 * sqrt 2)
            (-5 * 4)sqrt 2  ++  (2 * 7) * sqrt 2
           -20 * sqrt 2  ++  14 * sqrt 2
           -6 * sqrt 2 

    - Simplify more complicated radical expressions 
      ex: sqrt 9x  ++  sqrt 49x  -- sqrt 25x
          3 ++ 7 -- 5 * sqrt x
          5 * sqrt x  
          
      ex02: sqrt 6x^2  ++  x * sqrt 24
            x * sqrt 6  ++ x * sqrt 4 * 3 * 2
            x * sqrt 6  ++ 2x * sqrt 6
            3x * sqrt 6

      ex03: 3* sqrt 8x^2  --  4x* sqrt 2  --  x* sqrt 8
            3x* sqrt 4* sqrt 2  --  4x* sqrt 2  --  x* sqrt 4 * sqrt 2
            (3x * 2)* sqrt 2  --  4x* sqrt 2  --  2x* sqrt 2
            (6x - 4x - 2x)* sqrt 2
             0 * sqrt 2  
             0   


16.04 Rationalizing the Denominator

    - Rationalize denominators with square roots
         ex: 2 / sqrt 3 
             2 * sqrt 3 / sqrt 9
             2* sqrt 3 / 3

         ex02: 24* sqrt 10 / 16* sqrt 3
               24* (sqrt 10 * sqrt 3) / 16 * 3
               24* (sqrt 30) / 48
               24* sqrt 30 / 48
               sqrt 30 / 2   
               
         ex03: -3 / sqrt 50 
               -3 / sqrt 25 * (sqrt 2)^2
               -3* sqrt 2 / 5 * 2
               -3* sqrt 2 / 10
             
    - Write radicals in simplified form
         Simplified Form of a Radical
          1) the radicand has no factor (except 1) that is a perfect root (when dealing with square roots), a perfect cube (when dealing with cube roots), and so on...
          2) the radicand has no fractions
          3) no denominator contains a radical
          
         ex: sqrt 24 / sqrt 8
             sqrt 4 * sqrt 6  /  sqrt 4 * sqrt 2
             2* sqrt 6   /  2 * sqrt 2
             2* sqrt 12  /  2 * (sqrt 2)^2
             2* (sqrt 4 * sqrt 3)  / 4
             4 sqrt 3 / 4
             sqrt 3
         
         ex02: sqrt 1/2 
               sqrt 2 / 2

         ex03: sqrt 21 / sqrt 7  **  sqrt 21 / sqrt 8
               sqrt 21 * sqrt 21  //  sqrt 7 * sqrt 8
               sqrt 441  /  sqrt 56   
               21 / sqrt 4 * sqrt 14
               21* sqrt 14 / 28
               (3* sqrt 14) / 4

         ex04: sqrt 1/12  *  sqrt 1/3
               sqrt 1 * sqrt 1  **  sqrt 12 / sqrt 3
               sqrt 1 / sqrt 36
               1 / 6

    - Rationalize denominators with cube roots
         ex: 3-rt 4  /  3-rt 7
             3rt 4 * 3rt 49  //  3-rt 7 * 3-rt 49
             3rt 196  //  3-rt 343
             3rt 196  // 7


16.05 More Simplifying and Operations with Radicals

    - Simplify products of radical expressions
       Rules for Simplifying Radical Expressions
          1) if a radical represents a rational number, use that rational number in place of the radical
          2) if a radical expression contains products of radicals, use the product rule to get a single radical
                nth-rt X * nth-rt Y  =  nth-rt XY
          3) if a radicand has a factor that is a perfect square, express the radical as the product of the positive
             square root of the perfect square and the remaining radical factor. A similar statement applies to higher roots
          4) if a radical expression contains sums or differences of radicals, use the distributive property to combine
             like radicals
          5) rationalize any denominator that contains a radical

      ex: sqrt 5 * (sqrt 3 - sqrt 7)
          (sqrt 5 * sqrt 3) - (sqrt 5 * sqrt 7)
          sqrt 15 - sqrt 35

      
      ex02: 3* sqrt 14 * sqrt 2 - sqrt 28
            3* sqrt 28 - sqrt 28
            2* sqrt 28 
            2* sqrt 4 * sqrt 7
            2 * 2 * sqrt 7
            4* sqrt 7

       ex03: ((2* sqrt 6) + 3)((3* sqrt 6) + 7)  use FOIL
             (2 * sqrt 6 * 3* sqrt 6) 
             + 2 * sqrt 6 * 7
             + 3 * 3*sqrt 6 
             + 3 * 7

             6* (sqrt 6)^2
             + 14* sqrt 6
             + 9* sqrt 6
             + 21
             23* sqrt 6  +  57 

       ex04: (5 - sqrt 2)(5 + sqrt 2)
             25 - 2
             23     
             
       ex05: (2* sqrt 7 + 3)^2
             4*7  +  2*(2*3)* sqrt 7  +  9
             28 + 12* sqrt 7  +  9
             12* sqrt 7 + 37        

    - Use conjugates to rationalize denominators of radical expressions
       ex: 5 / 3 + sqrt 5
           5* (3 - sqrt 5) / (3 + sqrt 5)(3 - sqrt 5)
           5* (3 - sqrt 5) / 3^2 - (sqrt 5)^2
           5* (3 - sqrt 5) / 9 - 5
           5* (3 - sqrt 5) / 4

       ex02: 6 + sqrt 2  // sqrt 2 - 5
             (6 + sqrt 2)(sqrt 2 + 5)  //  (sqrt 2 - 5)(sqrt 2 + 5)
             6* sqrt 2 + 30 + 2 + 5* sqrt 2   /  2 - 25
             -32 - 11*sqrt 2) //  23 

    - Write radical expressions with quotients in lowest terms
        ex: 6*sqrt 11 - 12 // 6
            6* (sqrt 11 - 2)  // 6
            (sqrt 11) - 2

        ex02:  12 - sqrt 40  //  4
               12 - sqrt 4 * sqrt 10
               12 - 2* sqrt 10  // 4 
               2(6 - sqrt 10)  // 4
               (6 - sqrt 10) // 2


16.07 Rational Exponents

    Part A: nth Roots and Rational exponents
    nth root -- the index denotes the radical root of the radicand
        ex: 3-rt 125; 3 is the index that tells me i'm looking for a cubed root
            = 5

        if the nth root is positive then the principle root will be returned (positive absolute value of the answer)

        ex02: 4-rt m^4
              |m|         -- since this is an even root it will return the principle, absolute power, of the answer

        ex03: nth-rt A  ==  A^1/n -- is the way we can solve for nth roots
              (4/25)^1/2    == will equal  sqrt 4 / sqrt 25
              sqrt 4 = 2; sqrt 25 = 5
              answer = 2/5

        ex04: A^m/n  = nth-rt A^m  == (nth-rt A)^m
              8^4/3  = (3-rt 8)^4  -- can be factored
              3-rt 8^3 * 3-rt 8
              8 * 3-rt 8
              8 * 2 = 16 


        ex05:  rewrite with a rational exponent
               (5-rt 3x)^2      
               (3x)^2/5                     

    Part B: Simplify Expressions using the law of exponents
        zero exponent rule         a^0       == 1           -- if a != 0
        negative exponenet rule    a^-2      == 1/a^2       -- if a != 0                    
        Product Rule               a^2 * a^3 == a^(2 + 3)                           
        Quotient Rule              a^3 / a^2 == a^(3 - 2) == 1/a^(2-3)                           
        Power Rule                 (a^3)^2   == a^(3 * 2)                           
        Product to Power Rule      (a * b)^3 == a^3 * b^3                           
        Quotient to Power Rule     (a / b)^4 == a^4 / b^4  -- if b != 0                          
        Quotient to Negative Power (a/b)^-4  == (b/a)^4    -- if a & b != 0    
    
     When Simplifying:  
            1) ALL exponents are positive
            2) Each base only occurs once
            3) There are NO parenthesis in the expression
            4) There are NO powers written to powers  
            
     ex: x^1/4  //  x^5/6
         = x^(1/4 - 5/6)
         = x^ (3/12 - 10/12)
         = x^-7/12
         = 1/x^7/12

     ex02:  [(x^2/3 * y^-1/3) // (8x^1/2 * y)]^1/3    -- start in the parenthesis
            [(x^(2/3 - 1/2)) (y^-1/3) // (8*y )]^1/3  -- using the quotient rule we factor out the x from the denominator
            [(x^1/6) // (8* y^1 - -1/3)]^1/3          -- using the quotient rule we factor y from the numerator -- when factoring into the denoinator it is subtracted FROM the denominator  
            [(x^1/6)  // (8* y^4/3)]^1/3 
            (x^1/6 * 1/3) // (8^1/3 * y^4/3 * 1/3)    -- distribute the 1/3 exponent into ALL bases & multiply due to power rule
            x^1/18 // 8^1/3 * y^4/9                   -- take care of 8, 8^1/3 is the same as the 3-rt 8 which is a perfect cube and is 2
            x^1/18 // 2y^4/9

     ex03: sqrt x  //  4-rt x                         -- easiest to give rational exponents
           x^1/2   //  x^1/4                          -- raise the base by an exponent with the nth root as the denominator of the exponent
           x(^1/2 - ^1/4)
           x(^4/8 - ^2/8)
           x^1/4

     ex04: simplify by factoring out x^1/2
           2x^3/2  +  3x^1/2(x + 5)
           x^1/2(2x + 3(x + 5))
           x^1/2(2x + 3x + 15)
           x^1/2(5x + 15)
           ??? 5x^1/2(x + 3)   ---  i have verified that this is an 'Identity' of the above and is the same answer

     ex05: factor out 2(x - 4)^-1/3
           2(x - 4)^-1/3 * (4x - 3) + 12(x - 4)^2/3  
           2(x - 4)^-1/3[1 * (4x - 3) + 6(x - 4)]           -- factoring out the 2(x-4)^-1/3
           2(x - 4)^-1/3(4x - 3 + 6x - 24)        
           2(x - 4)^-1/3(10x - 27)
