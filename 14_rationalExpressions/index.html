14.1 the Fundemental Property of Rational Expressions

         - Find the values of the variable for which a rational a rational expression is undefined
                rational expression is an expression of the form P/Q, where P and Q are polynomials, with Q != 0
             
            ex: 4x/x^2 + 4; 6m/11; (3p^2 - 6p + 9)/p - 3
            
         - Find the numerical value of a rational expression
            ex: find numerical value of (-3x)^2 / (4x + 12)
                    if x = 2
                (-3 * 2)^2 / (4 * 2 + 12)
                (-6)^2  / 20
                36 / 20  == 9/5

            ex02: (-3x)^2 / (4x + 12) if x = -3
                  (-3 * -3)^2  / (4 * (-3) + 12)
                  (9)^2 / 0    == undefined 81/0 is undefined (anything divided by 0 is undefined)
