14.1 the Fundemental Property of Rational Expressions

         - Find the values of the variable for which a rational a rational expression is undefined
                rational expression is an expression of the form P/Q, where P and Q are polynomials, with Q != 0
             
            ex: 4x/x^2 + 4; 6m/11; (3p^2 - 6p + 9)/p - 3
            
         - Find the numerical value of a rational expression
            ex: find numerical value of (-3x)^2 / (4x + 12)
                    if x = 2
                (-3 * 2)^2 / (4 * 2 + 12)
                (-6)^2  / 20
                36 / 20  == 9/5

            ex02: (-3x)^2 / (4x + 12) if x = -3
                  (-3 * -3)^2  / (4 * (-3) + 12)
                  (9)^2 / 0    == undefined 
                  81/0         is undefined (anything divided by 0 is undefined)

         - Write rational expressions in lowest terms
                something is in 'lowest terms' if the greatest common factor of the numerator and denominator is 1

            ex:          (2x^2 - 3x - 5) / (2x^2 - 7x + 5)  
            factor first (2x - 5)(x + 1) / (2x - 5)(x - 1) 
            cancel out    (x + 1) / (x - 1)

        ** caution ** you MUST factor the polynomials first before reducing to lowest terms 

            ex02:        (m^2 - n^2) / (m + n)
            factor       (m + n)(m - n) / (m + n)
            cancel out   m - n

            ex03:        (6 - t) / (t - 6)
            factor       -1(t - 6) / (t - 6)
            cancel       -1 is the answer

        ** caution ** you cannot cancel out terms from the rational expression, only FACTORS 
             you must first factor the polynomials and reduce from there
         
         - Recognize equivalent forms of rational expressions
            -(x + 3) / 2
            -x -3 / 2
            x + 3 / -2 
              
            all of these are equivalent -- you cannot distribute a negative sign to both the numerator and denominator 

            (-x - 3) / -2  != -(x + 3) / 2

        ex:          -(x + 4) / (x - 3)
        is equal to  (-x -4) / (x - 3)
        also equals  (x + 4) / (-x + 3)
    

14.2 Multiplying and Dividing Rational Expressions

         - Multiplying Rational Expressions
                the product of the rational expression p/q * r/s
                 is equal to pr/qs
            
            ex: 16y^4 / 18y^5 * 15y^5 / y^2 
                240y^9 / 18y^7
                40y^2 / 3

            ex01: 2(c + d) / 3 * 18 / 6(c + d)^2
                  2 * 18 (c + d) / 3 * 6 (c + d)^2
                  36 (c + d) / 18 (c + d)^2
                  2 / (c + d)                 

         - Find Reciprocals
            ex: (r^2 + rp) / 7
                7 / (r^2 + rp)

            ex02: (z2 + 7z + 12) / z^2 - 9
                (z^2 - 9) / (z^2 +7z + 12)
            
         - Divide Rational Expressions
                division = mulitiplication of the reciprocal

            if p / q and r / s are 2 rational expressions, with r / s != to 0,
             then p/q / r/s = p/q * s/r = ps / qr
             
            ex: 9z^4 / 3z^5 // 3z^2 / 5z^3
                9z^4 / 3z^5 * 5z^3 / 3z^2
                9z^4 * 5z^3 // 3z^5 * 3z^2
                45z^7 // 9z^7
                5

            ex02: (2k^2 - k - 1) / (2k^2 + 5k + 3) // (4k^2 - 1) / (2k^2 + k - 3)
                  (2k^2 - k - 1) / (2k^2 + 5k + 3) * (2k^2 + k - 3) / (4k^2 - 1)
                  (2k^2 - k - 1) * (2k^2 + k - 3) // (2k^2 + 5k + 3) * (4k^2 -1)
                  (2k + 1)(k - 1) * (2k +3)(k - 1) // (2k _+ 3)(k + 1) * (2k + 1)(2k - 1)
                  (k - 1)^2 // (k + 1)(2k - 1)  

            Rules for Multiplying or dividing Rational expressions
                 1) Note the Operation - if the operation is division then switch and multiply by the reciporcal
                 2) Factor - factor numerators and denominators completely
                 3) Multiply - multiply numerator by numerator and denominator by denominator
                 4) Simplify - write the answer in lowest terms

            ex03: (p^2 + 4p - 5) / (p^2 + 7p + 10) // (p - 1) / (p + 4)
                  (P^2 + 4p - 5) * (p + 4) // (p^2 + 7p + 10) * (p - 1)
                  (p - 1)(p + 5) * (p + 4) // (p + 5)(p + 2) * (p - 1)
                  (p + 4) // (p + 2)
         

14.3 Least Common Denominator

        - Find the least common denominator for a group of fractions
            smallest number that all denominators factor into
            Finding Least Common Denominator
            1) Factor - factor each denominator into prime factors
            2) List the Factors - list each different denominator factor the greates number of times it appears in any of the denominators
            3) Multiply - multiply the denominator factors together to get the LCD

           ex:  6/21r^3,  8/12r^5
                21r^3 = 3 * 7 * r^3
                12r^5 = 2 * 2 * 3 * r^5
                LCD = 2 * 2 * 3 * 7 * r^5 = 84r^5

            ex02: 7 / (5b - 10) ; 11 / (6b - 12)
                  7 / 5(b - 2) ; 11 / 6(b - 2)
                  = 6 * 5 (b - 2)
                  LCD = 30(b - 2)

            ex03: 3 / (k^2 + 5k) ; 2 / (k^2 + 3k - 10)       
                  k(k + 5) ; (k + 5)(k - 2)
                  LCD = k(k+5)(k-2)

        - Rewrite rational expressions with given denominators
            ex: 4 / 11 = ? / 55
                you have to multiply 11 * 5 in order to get 55
                20 / 55

            ex02: 8 / (3x + 1) = ? / (12x + 4)    
                  multiply (3x + 1) by 4 to get (12x + 4)
                  32 / (12x + 4)

            ex03: 12p / (p^2 + 8p) = ? / (p^3 + 4p^2 - 32p) 
                  (p^3 + 4p^2 - 32p) = p(p^2 + 4p - 32) = p(p - 4)(p + 8)
                  12p(p - 4) / p (p-4)(p+8)

            ex04: 14 / (z^2 - 3z) = ? / z(z - 3)(z - 2)
                  z(z - 3) = z(z - 3)(z - 2)
                  14 (z - 2) / z(z - 3)(z - 2)
                  14z - 24 / z(z - 3)(z - 2)
        

14.4 Adding and Subtracting Rational Expressions

        - Add rational expressions having the same denominator
            if P/Q and R/Q (Q != 0) are rational expressions,
                then P/Q + R/Q =  (P + Q) / R
            
            ex: x^2 / (x + 5) + 5x / (x + 5)
                x^2 + 5x / (x + 5)
                x(x + 5) / (x + 5)
                x     

        - Add rational expressions having different denominators
            Rules for Adding with Different Denominator 
            1) Find the LCD - find the least common denominator.   
            2) Rewrite fractions - rewrite rational expressions as an equivalent rational expression with the LCD as the denominator.   
            3) Add - add the numerators to get the numerator of the sum. The LCD is the denominator of the sum.    
            4) Write in lowest terms - use the fundemental property to write the answer in lowest terms.   

            ex: z/5 + 1/3
                3z/15 + 5/15
                (3z + 5) / 15    

            ex02: x/(x - 2)  +  4/(x + 2)
                  x(x + 2) + 4(x - 2) / (x + 2)(x - 2) 
                  (x^2 + 2x) + (4x - 8) // (x + 2)(x - 2)
                  x^2 + 6x - 8 // (x + 2)(x - 2)
                  
            ex03: (x + 3y) / (x^2 + 2xy + y^2) + (x-y) / (x^2 + 4xy + 3y^2)
                  denominator = (x + y)(x + y) && (x + 3y)(x + y)
                  LCD = (x+y)^2 * (x + 3y)
                  (x + 3y)^2 / (x+y)^2 * (x+3y) ++ (x-y)(x+y) / (x + y)^2 * (x + 3y)   
                  (x + 3y)^ 2 + (x-y)(x + y) // (x + y)^2 * (x + 3y)
                  (x^2 + 6xy + 9y^2) + (x^2 - y^2) // (x + y)^2 * (x + 3y)
                  (2x^2 + 6xy + 8y^2) // (x + y)^2 * (x + 3y)
 
        - Subtract rational expressions
            
            ex:  3x / x - 5  - (2x - 25) / 5 - x
                 LCD = (x - 5)
                 3x / (x - 5) -  (-2x + 25) / (x-5)
                 3x - (-2x + 25) // (x - 5)
                 5x - 25 / (x - 5)
                 5(x - 5) / (x - 5)
                 5

            ex02: 6x / (x^2 - 2x + 1)  -  1 /(x^2 - 1)
            denom: (x -1)(x - 1)  && (x + 1)(x - 1)
             LCD: (x - 1)^2 * (x + 1) 
                  6x(x + 1) - (x - 1)  // (x-1)^2 * (x + 1)
                  6x^2 + 5x + 1
                  (2x + 1)(3x + 1) // (x - 1)^2 * (x + 1)                  

            ex03:  5 / (x^2 - 9) - (x + 2) / (x^2 + 4x + 3)
            denom: (x + 3)(x - 3) && (x + 3)(x + 1)
             LCD : (x + 3)(x - 3)(x + 1)
                   5(x + 1) - (x + 2)(x - 3) // (x + 3)(x - 3)(x + 1)  
                   5x + 5 - (x^2 - x - 6)  // (x + 3)(x - 3)(x + 1)     
                   (5x + 5) - x^2 + x + 6
                   -x^2 + 6x + 11 // (x + 3)(x - 3)(x + 1)
